# Advanced Gameplay Object Manager

Complex system for spawning and managing gameplay objects with integrated physics and gravitational interactions.

## Key Features
- **Multi-Object Spawning**: Manages spaceships, asteroids, planets, black holes, and space stations
- **Gravity System**: Custom gravitational calculations between celestial bodies
- **Object Pooling**: Performance-optimized spawning with lifecycle management
- **Event-Driven Architecture**: Coroutine-based spawning with random intervals

## Technical Highlights
- Physics-based gravitational forces between all object types
- Intelligent spawn positioning around player
- Warning system for special events (black holes, space stations)
- Optimization through distance-based object management
- Support for different object categories with individual spawn rules

# Advanced Player Control System

Comprehensive player controller featuring multiple input methods, physics-based movement, and complex game mechanics.

## Key Features
- **Multiple Control Schemes**: Tap buttons, slider joystick, and gyroscope support
- **Physics-Based Movement**: Realistic acceleration and velocity management
- **Health & Shield Systems**: Damage calculation and shield management
- **Procedural Mesh Generation**: Dynamic asteroid shape creation
- **Upgrade System**: Ship progression with 12 different levels

## Technical Highlights
- Custom input handling for different control modes
- Complex collision detection and damage calculation
- Particle system integration for visual effects
- Economy system with coin collection and spending
- Save/load system for player progress
