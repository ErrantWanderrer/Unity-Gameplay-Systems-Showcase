# Core Game Management System

Central controller managing game state, UI, settings, and system integration for the entire application.

## Key Features
- **Game State Management**: Comprehensive state control (menu, gameplay, pause, etc.)
- **UI System**: Complete user interface management and animation
- **Settings Controller**: Graphics, audio, and gameplay settings
- **Store Integration**: In-app purchases and upgrade systems
- **Analytics & Tracking**: Player progress and statistics monitoring

## Technical Highlights
- Post-processing effects control (Bloom, Chromatic Aberration, Vignette)
- Audio system with dynamic music and sound effects
- Save/Load system using PlayerPrefs
- Mobile optimization settings (performance modes, graphics quality)
- Cross-platform input handling
